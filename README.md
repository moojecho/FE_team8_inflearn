# 구현 기능

## 메인페이지

- 배너
- 캐러셀
- 전체 강의 불러오기

## 검색페이지

- 강의 검색

## 로그인/회원가입 페이지

- 로그인
- 회원가입
- 카카오 소셜 로그인

## 강의 상세페이지

- 강의 상세정보
- 수강평 CRUD
